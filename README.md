## scrimBot

### Description
When torrents aren't an option, bring it ON =).

![scrimBot](/screenshots/scrimBot.png "scrimBot")

### Local setup

Global packages:
- [`nodemon`](https://github.com/remy/nodemon)
- [`node (verion 7 or higher)`](https://nodejs.org/en/download/package-manager/)

```javascript
$ npm install
```
